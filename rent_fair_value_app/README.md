
# 전월세 매물 경제성 비교 앱

실매물의 보증금, 월세, 관리비를 같은 기준의 **월 환산 총비용**으로 비교하는 Streamlit 앱입니다.

## 실행 방법

터미널에서 아래 명령어를 실행하세요.

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 입력 단위

모든 금액은 **만원 단위**입니다.

예:
- 보증금 1,000만 원 → `1000`
- 월세 60만 원 → `60`
- 관리비 10만 원 → `10`

## 계산 공식

월 환산 총비용 = 월세 + 관리비 + 보증금 기회비용 + 보증금 대출이자

이 앱은 특정 전세 하나를 기준으로 비교하지 않고, 모든 실매물을 같은 금융 기준으로 환산해서 비교합니다.
